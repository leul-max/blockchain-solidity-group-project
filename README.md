## 💻 Smart Contracts Implemented

### 1. Simple Storage
- Basic variable types
- Functions (store/retrieve)
- Arrays and mappings

### 2. Storage Factory
- Deploying contracts from another contract
- Managing multiple contract instances

### 3. FundMe
- Sending and receiving ETH
- msg.sender and msg.value usage
- Require and custom errors
- Function modifiers (onlyOwner)
- Constructor and immutability
- Price conversion using libraries

## 🚀 How to Run the Code
1. Open Remix IDE (https://remix.ethereum.org)
2. Upload `.sol` files from `/code`
3. Compile the contracts
4. Deploy using JavaScript VM or testnet
5. Interact with contract functions
