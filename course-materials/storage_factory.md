# Storage Factory

## Topics Covered
- Contract Deployment
- Contract Interaction

## Key Concepts

### 1. Deploying Contracts from Contracts
- One contract can create another contract using `new`

### 2. Storing Contract Instances
- Use arrays to store deployed contracts

### 3. Interacting with Contracts
- Call functions of deployed contracts

## Example Idea
A factory contract creates multiple storage contracts and interacts with them.

## Summary
StorageFactory shows how contracts can manage and communicate with other contracts.
