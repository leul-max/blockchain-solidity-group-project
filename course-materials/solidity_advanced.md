# Solidity Advanced Concepts

## Topics Covered
- Solidity Math
- SafeMath
- For Loops
- Resetting Arrays
- Sending ETH from Contract

## Key Concepts

### 1. Solidity Math
- Handles arithmetic operations
- Solidity 0.8+ has built-in overflow protection

### 2. SafeMath
- Prevents overflow/underflow (used in older versions)

### 3. For Loop
- Used to iterate over arrays

### 4. Resetting Arrays
- Example: array = new uint[](0);

### 5. Sending ETH
- transfer()
- call()
- send()

## Summary
Advanced concepts improve contract efficiency, safety, and control.
